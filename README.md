
# BrightInk Kids

BrightInk Kids is a React + Vite app packaged for Android and iOS with Capacitor.

## Running the Code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

Run `npm run build` to create the web build in `dist`.

Run `npm run cap:sync` after a web build to sync the compiled app into the native Capacitor projects.
